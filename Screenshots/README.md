# Screenshots

This folder contains visual evidence collected during testing.

- **/Defects/** — Screenshots showing confirmed bugs (named by Bug ID, e.g., `BUG-001.png`)
- **/Observations/** — Screenshots of general observations, UI states, or reference captures used to support suggestions

**Naming convention:** `BUG-XXX.png` for defects, `OBS-XXX.png` for general observations.
